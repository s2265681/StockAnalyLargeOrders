# NiuNIuNiu 项目分析与打板优化建议

## 一、核心功能

**NiuNIuNiu** 是一个专注于 **L2 大单资金流分析** 的股票看盘系统，前端 React + 后端 Flask 架构。

**已实现功能：**

- **多源数据聚合**：东方财富(免费/L2)、新浪、腾讯、akshare、efinance，带优先级切换和降级
- **大单分级追踪**：D300(≥300万)、D100(≥100万)、D50(≥50万)、D30(≥30万)，区分买卖方向
- **分时图可视化**：价格线 + 均价线 + 成交量柱，240 分钟完整交易轴
- **盘口语言识别**：五档压单/托单/夹单震荡等信号检测，带置信度评分
- **涨停板块聚合**：按题材分组展示当日涨停股
- **数据模拟回放**：支持历史数据按时间轴回放，多种速度

## 三、当前需要优化的点

### 架构层面

1. **缺少 WebSocket**：当前靠轮询(10秒)获取实时数据，延迟大、资源浪费。盘口语言判断对时效性要求高，建议改 WebSocket 推送
2. **缓存策略粗糙**：当日数据 5 秒缓存、历史 5 分钟，但盘中快速行情 5 秒仍然太长，尤其打板决策是秒级的
3. **无持久化存储**：所有数据内存缓存，重启丢失。历史大单数据应入库(SQLite/PostgreSQL)供回测

### 数据层面

4. **免费数据源延迟**：东方财富免费接口有 3-5 秒延迟，对打板来说可能错过最佳时机
5. **大单识别仅按金额**：缺少拆单识别（机构大单拆成多个小单），应加入时间窗口内的聚合分析
6. **缺少逐笔委托数据**：只有成交数据，没有撤单数据，无法判断"假封板"

### 前端层面

7. **盘口语言信号较简单**：只有压单/托单/夹单等基础判断，缺少量化评分体系
8. **无声音/桌面通知**：打板关键信号应有即时告警

## 四、打板选手建议增加的指标

### 4.1 核心指标（强烈建议）

| 指标 | 说明 | 实现思路 |
|------|------|---------|
| **封单量/流通盘比** | 封板力度的核心，封单金额占流通市值比例越大越安全 | 买一价=涨停价时，累计买一量 / 流通盘 |
| **封单变化趋势** | 封板后封单是增加还是减少，减少预示炸板风险 | 每秒采样封单量，计算斜率 |
| **涨停时间点** | 越早涨停越强势（9:25竞价涨停 > 10:00前 > 午后） | 记录首次触板时间 |
| **炸板次数** | 炸板后回封说明分歧大，多次炸板胜率骤降 | 统计价格触及涨停价后回落的次数 |
| **连板高度/梯队** | 当日市场最高连板数、断板率，判断情绪周期 | 统计全市场连板股，构建梯队图 |

### 4.2 重要辅助指标

| 指标 | 说明 |
|------|------|
| **竞价量能** | 9:15-9:25 集合竞价成交量和价格变化，量缩价升是强势信号 |
| **分时量比** | 当前成交量 / 过去5日同时段均量，>2 说明资金异动 |
| **板块联动度** | 同题材涨停/涨幅前列股票数量，独狼涨停风险高 |
| **换手率到板** | 涨停时的换手率，过高(>15%)说明抛压大 |
| **大单净买占比** | 大单净买入金额 / 总成交额，>30% 说明主力坚决 |
| **委比/委差变化** | 盘口买卖委托比例的动态变化 |

### 4.3 进阶指标（如有 L2 数据）

| 指标 | 说明 |
|------|------|
| **逐笔大单拆单识别** | 10秒内同方向多笔中单可能是拆单，合并计算真实大单 |
| **撤单率** | 涨停价撤单量/挂单量，高撤单率=假封板 |
| **主力单比例** | 500手以上超大单占比，区分游资和机构 |
| **竞价抢筹检测** | 9:20后大额买单不撤，说明资金坚决 |
| **尾盘封单结构** | 14:30后封单是否由少数大户维持 |

### 4.4 情绪面指标

| 指标 | 说明 |
|------|------|
| **市场涨停家数/跌停家数比** | 大盘情绪温度计 |
| **昨日涨停今日表现** | 昨日涨停股今日涨跌分布，衡量接力意愿 |
| **市场高低切换信号** | 高位股补跌+低位股启动=风格切换 |
| **龙虎榜预判** | 实时估算当日可能上龙虎榜的资金结构 |

## 五、优先实现建议

如果要快速提升打板胜率，建议优先做这 3 件事：

1. **封单监控模块**：封单量、封单变化率、炸板次数 —— 这是打板最核心的决策依据
2. **板块联动看板**：当前题材涨停数量+跟风力度，避免追独狼
3. **竞价分析**：9:15-9:25 的量价分析，是判断一字板和换手板的关键窗口

这些数据大部分可以通过现有的东方财富免费接口获取，改动成本相对可控。
