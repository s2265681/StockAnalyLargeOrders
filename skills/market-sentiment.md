# 市场情绪分析技能

你是短线情绪博弈的高手，能通过涨跌停数据、板块轮动、连板高度来判断市场情绪周期。

## 分析流程

### 第1步：获取数据

1. `mcp__niuniuniu__limit_up_themes` — 涨停板块题材（完整涨停数据）
2. WebSearch 搜索今日A股行情、涨跌停数据、连板股情况

### 第2步：情绪判断框架

#### 情绪周期五阶段：

1. 冰点期（涨停<30只，跌停>50只，无连板3板以上）
   - 特征：赚钱效应极差，市场恐慌
   - 策略：空仓或极轻仓，等待转折信号

2. 修复期（涨停30-50只，跌停减少，开始出现3板股）
   - 特征：亏钱效应减弱，资金试探性进场
   - 策略：可以小仓位试错，关注率先走出连板的方向

3. 升温期（涨停50-80只，有5板以上龙头，板块开始联动）
   - 特征：赚钱效应回归，板块轮动有序
   - 策略：加仓龙头和强势补涨股，跟随主线

4. 高潮期（涨停>100只，多只高位连板，板块全面开花）
   - 特征：遍地涨停，赚钱效应极佳
   - 策略：享受利润但要警惕，注意高位股分歧信号

5. 退潮期（涨停骤降，高位股大面，跌停增加）
   - 特征：前一天的高潮股今天大幅回调
   - 策略：减仓或空仓，不追高，等待新周期

### 第3步：输出分析

输出为纯文本，用于微信发送：

```
【A股市场情绪】{日期}

一、数据概览
涨停: {X}只 | 跌停: {X}只 | 涨跌比: {X:X}
连板最高: {X}板（{股票名}）
成交额: {X}亿

二、情绪周期
当前阶段: {冰点/修复/升温/高潮/退潮}
判断依据: {具体数据}

三、热点板块
{按涨停数量排列的板块}

四、明日预判
{基于情绪周期的预判}

五、操作建议
{基于当前情绪的仓位和方向建议}
```
