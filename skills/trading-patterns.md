# 短线核心战法

三大实战战法，每个战法都有严格的触发条件、操作纪律和止损规则。

## 战法一：龙头首阴低吸

### 核心逻辑
龙头股在连续上涨后首次收阴线（首阴），是强势调整而非趋势反转。市场资金对龙头有记忆和惯性，首阴往往是主力洗盘或正常换手，次日大概率修复。

### 触发条件（缺一不可）
1. **龙头确认**：该股必须是板块/市场公认的龙头（连板最高、辨识度最强、资金认可度最高）
2. **首次收阴**：连续涨停或大涨后的**第一根**阴线（第二根阴线不算首阴）
3. **阴线质量**：
   - 跌幅在-3%到-7%之间最佳（太小没洗干净，太大可能是真出货）
   - 成交量相比前一日不能放大超过2倍（放巨量阴线=出货信号）
   - 不能是一字跌停或接近跌停的大阴线
4. **情绪环境**：市场情绪不在极端退潮期（退潮期龙头首阴大概率是断头铡刀）
5. **板块支撑**：所在板块当日没有崩塌（板块跌幅<3%）

### 买点
- **最佳买点**：首阴日尾盘30分钟，股价缩量企稳不再创新低时
- **次优买点**：次日低开不破首阴最低价，分时企稳回升时
- **激进买点**：首阴日盘中跌到-5%~-7%缩量时（需要胆量）

### 仓位
- 升温期/高潮期：3-4成
- 修复期：2成
- 退潮期：不做或极轻仓（0.5-1成）

### 止损
- 无条件止损：跌破首阴最低价的3%
- 时间止损：买入后2个交易日内未修复（不涨反跌）则止损
- 情绪止损：市场情绪急剧恶化（跌停数>涨停数）时无论盈亏都出

### 目标
- 保守目标：修复到首阴开盘价附近（+5%~8%）
- 乐观目标：突破前高，再创新高（板块二次启动时）
- 超预期：再次涨停封板（龙头反包，这是最强信号）

### 经典案例场景
- 6板龙头股次日大幅低开-5%收阴，但全天没有跌停，尾盘缩量企稳 → 次日高概率反包
- 市场总龙头连涨后首次分歧日收小阴线(-2%)，成交量萎缩 → 正常洗盘，可低吸

### 风险警示
- 高位龙头（8板以上）的首阴可能是天地板的前兆，更要严格止损
- 情绪退潮期的首阴成功率极低（<30%），不要逆周期硬做
- 放巨量阴线（量比>3）大概率是主力出货，不是首阴低吸机会

## 战法二：龙头爆量涨停 & 次日竞价高开弱转强

这是一个完整的两日战法：第一日识别龙头爆量涨停（选股），第二日通过竞价高开弱转强确认买点（择时）。核心思路是"今天选股，明天确认"。

### 第一日：龙头爆量涨停（选股阶段）

#### 核心逻辑
龙头股在关键位置（突破前高/板块启动日/情绪修复日）放巨量封涨停，说明大资金强势抢筹，分歧转一致的过程中资金用真金白银投票。爆量涨停=资金共识+筹码充分换手，后续空间往往更大。

#### 触发条件（缺一不可）
1. **龙头地位**：板块龙头或市场辨识度前三的标的
2. **爆量标准**：
   - 成交额创近10日新高
   - 换手率>10%（中小盘）或 >5%（大盘股50亿以上流通市值）
   - 量比>2（相对前5日均量）
3. **涨停封板**：必须封死涨停（烂板不算，反复开板超3次不算）
4. **封板时间**：14:00前封板（尾盘封板的爆量更可能是资金分歧而非共识）
5. **板块配合**：板块当日有3只以上涨停（有板块效应支撑）

#### 关键区分：好的爆量 vs 坏的爆量
- **好的爆量涨停**：前期缩量调整后首次放量涨停、低位突破平台放量涨停、板块启动日龙头率先放量涨停
- **坏的爆量**：连续涨停后成交额创天量但收长上影/没封住 = 见顶信号；高位放量涨停但封单持续缩小 = 对倒出货

#### 当日操作选择
- **打板买入（激进）**：封板瞬间排队打板，或开板回封时介入。仓位3-5成
- **不买入，等次日竞价确认（稳健，推荐）**：收盘后将该股加入次日竞价观察池

### 第二日：竞价高开弱转强（确认阶段）

#### 核心逻辑
头天爆量涨停的龙头股，次日集合竞价阶段高开（+2%以上），但开盘后短暂回落（弱），随后快速拉升突破开盘价（转强）。这个"弱转强"过程说明：高开时的抛压被消化，新资金接力进场，是确认性极强的买入信号。

#### 竞价阶段观察（9:15-9:25）

**第一阶段（9:15-9:20）：自由竞价**
- 这阶段可以撤单，参考价值不高
- 但如果开盘价持续上移且量能放大 → 积极信号
- 如果虚假高开后快速回落 → 可能是假竞价

**第二阶段（9:20-9:25）：不可撤单（核心观察期）**
- **竞价量**：占前一日成交量的比例
  - >3% = 强竞价（资金抢筹意愿强）
  - 1-3% = 正常
  - <1% = 弱竞价（关注度不高）
- **竞价价格走势**：
  - 9:20后价格持续上移 = 强（买方持续加价）
  - 9:20后价格走平 = 中性
  - 9:20后价格回落 = 弱（卖方挂单多）
- **最终竞价涨幅**：
  - +5%以上：强势竞价，但追高风险大，需要更强的弱转强确认
  - +2%~5%：最佳区间，说明被认可但不至于一致性太强
  - 0%~2%：弱竞价，需盘中更明确的弱转强信号
  - 平开或低开：放弃该战法，可能转弱

#### 开盘后弱转强的判断

**什么是"弱"（开盘后的回落）**
- 开盘后5-10分钟内从高开价回落
- 回落幅度：不超过竞价涨幅的50%（如竞价+4%开盘，回落到+2%以上为正常）
- 回落时成交量萎缩（缩量回调=正常洗盘）
- 不能跌破昨日收盘价（跌破=转弱失败，放弃）

**什么是"转强"（回落后的拉升）**
- 开盘后10-30分钟内，价格重新突破开盘价
- 突破时成交量放大（放量突破=有效）
- 分时线站上分时均线
- **最强信号**：突破开盘价后直接封涨停

#### 买点
- **标准买点**：价格突破开盘价的瞬间（确认弱转强成立）
- **提前买点**：回落缩量止跌，分时开始拐头向上时（更好的价格，但不确定性更大）
- **保守买点**：突破开盘价后回踩不破分时均线时

### 仓位（综合两日判断）
- 高确定性（龙头+板块强+情绪升温+强竞价+标准弱转强）：4-5成
- 中确定性（龙头+情绪一般 或 竞价偏弱但弱转强成立）：2-3成
- 低确定性（弱竞价+回落较深+勉强转强）：1-2成

### 止损
- **弱转强失败**：突破开盘价后又跌回，且跌破回落最低点 → 立即止损
- **跌破昨收**：跌破昨日涨停价（昨日收盘价）→ 立即止损
- **绝对止损**：亏损超过-7%无条件出局
- **时间止损**：买入后60分钟内未创新高 → 减仓观察

### 目标
- 首板爆量涨停 → 次日弱转强买入后预期溢价5-8%
- 连板爆量涨停（加速板）→ 次日弱转强后继续涨停概率>50%（高潮期）
- **最强走势**：弱转强后直接封涨停，继续持有享受连板溢价

### 次日持股操作（如果头天打板买入的）
- **高开5%+且快速封板**：继续持有，享受溢价
- **高开5%+冲高不封板**：冲高时卖出1/2，留1/2等回封
- **平开或小高开(0-3%)**：观察前30分钟，向上突破均线持有，向下破均线减仓
- **低开(-3%以下)**：缩量低开等反弹减仓，放量低开直接止损

### 与情绪周期的关系
- **升温期**：弱转强成功率最高(70%+)，可以积极参与，仓位可以打满
- **高潮期**：一致性太强可能高开直接封板来不及买，但爆量涨停本身需谨慎（可能见顶）
- **修复期**：只做最强龙头的弱转强，仓位控制在2成
- **退潮期**：高开低走概率极大(>60%)，弱转强大多是假信号，不参与
- **冰点期**：不做

## 战法三：首板一进二

这是一个从当日涨停板中筛选次日最可能连板的标的，并在次日竞价/开盘确认后买入的战法。核心思路是"今天筛首板，明天做二板"。

### 核心逻辑
首板涨停是市场资金对某个方向的首次投票。并非所有首板都有二板潜力，关键在于筛选出"强首板"——封板质量高、题材辨识度强、资金共识明确的标的。次日通过竞价确认资金接力意愿后介入。

### 第一步：首板筛选条件

#### 硬性条件（缺一不可）
1. **题材辨识度**：必须属于当日市场主线/热点板块，板块内有3只以上涨停形成板块效应
2. **封板质量**：
   - 封板时间：10:30前封板为强（越早越好），10:30-13:00可接受，13:00后偏弱
   - 封板过程：一波流直接封板最强，3波内封板可接受，反复开板超3次排除
   - 不能是烂板（封板后多次打开又封回）
3. **量价关系**：
   - 换手率5%-15%为合理区间（<5%筹码没充分交换，>15%抛压太大）
   - 一字板排除（无换手，次日抛压集中释放风险大）
4. **市值区间**：流通市值50亿-200亿最佳（太小容易被控盘，太大资金拉不动）

#### 加分条件（越多越好）
1. **封单强度**：封成比（封单额/当日成交额）>3为强，>10为超强（次日高开概率>70%）
2. **分时形态**：60度角流畅上攻到板，无大幅回落
3. **位置优势**：低位突破平台首板 > 趋势中继首板 > 高位加速首板
4. **龙头属性**：板块内最早封板、最高辨识度、最强资金认可
5. **消息催化**：有明确政策/事件驱动（如行业政策、业绩超预期、重大合同）

#### 排除条件（有一条即排除）
1. 尾盘（14:30后）偷袭涨停
2. 一字板/T字板（无充分换手）
3. 非主流题材的独立涨停（无板块效应）
4. 近期有减持/利空公告
5. ST股或复牌补涨股

### 第二步：次日竞价确认（9:15-9:25）

#### 强竞价信号（次日二板概率高）
- 竞价成交额 > 前一日成交额的3%（资金抢筹意愿强）
- 高开幅度+3%~+5%（最佳区间，被认可但不至于一致性过强）
- 9:20后竞价价格持续上移（真金白银阶段，买方持续加价）
- 竞价量能放大至首板爆量的50%以上

#### 弱竞价信号（放弃或降低仓位）
- 竞价成交额 < 前一日成交额的1%
- 平开或低开
- 9:20后竞价价格回落
- 高开>7%（一致性太强，追高风险大，可能开板震荡）

### 第三步：开盘后买点

#### 买入方式一：竞价抢筹（激进）
- 条件：强竞价信号齐备 + 龙头地位 + 升温期
- 9:24-9:25挂单买入
- 风险：高开低走则被套在高位

#### 买入方式二：开盘确认（稳健，推荐）
- 开盘后观察5-10分钟
- **直接封板**：排队打板（最强走势，但可能买不到）
- **高开回落后弱转强**：回落不破昨日收盘价，放量突破开盘价时买入
- **高开震荡后拉升**：在分时均线上方横盘后向上突破时买入

#### 买入方式三：盘中二次封板（保守）
- 开盘冲高回落后，盘中再次拉升封涨停
- 二次封板确认资金态度坚决
- 适合风险偏好较低的选手

### 仓位
- 强首板+强竞价+升温期：3成
- 普通首板+正常竞价：1-2成
- 分散操作：同时关注2-3只首板，每只1成，分散风险
- **单票绝不超过3成**

### 止损
- **竞价买入**：开盘后跌破竞价成交价的3% → 立即止损
- **盘中买入**：跌破买入价的5% 或 跌破昨日涨停价 → 立即止损
- **时间止损**：买入后30分钟内未封板且走势转弱 → 减仓
- **绝对止损**：亏损>7%无条件出局

### 目标
- 核心目标：次日封住二板（+10%溢价）
- 保守目标：高开后获利3-5%了结
- 超预期：二板封死后继续持有博三板（仅限升温/高潮期的绝对龙头）

### 备选首板推荐逻辑
每日收盘后从当日涨停板中筛选备选池，按优先级排序：
1. **主线龙头首板**：当日最强板块中最早封板、辨识度最高的标的
2. **补涨龙首板**：主线板块中第二梯队，前排龙头已连板，补涨股首次涨停
3. **新题材首板**：全新题材/消息催化的首次涨停，板块有3只以上跟风
4. **趋势突破首板**：长期横盘后放量突破的首板，技术面和题材面共振

### 与情绪周期的关系
- **冰点期**：不做一进二，首板成功率极低
- **修复期**：只做绝对主线龙头首板，1成仓位试错
- **升温期**：一进二成功率最高(40%+)，积极筛选，可做2-3只分散
- **高潮期**：首板数量多但分歧加大，优选封板最早最强的，注意炸板风险
- **退潮期**：一进二成功率骤降(<20%)，不参与

| 战法 | 冰点期 | 修复期 | 升温期 | 高潮期 | 退潮期 |
|------|--------|--------|--------|--------|--------|
| 龙头首阴低吸 | 不做 | 极轻仓试错 | 标准仓位 | 标准仓位 | 不做 |
| 爆量涨停&弱转强 | 不做 | 只做龙头(2成) | 积极参与(4-5成) | 谨慎(可能见顶) | 不做(假信号多) |
| 首板一进二 | 不做 | 只做主线龙头(1成) | 积极筛选(2-3成) | 优选最强(注意炸板) | 不做(<20%成功率) |

## 分析流程

### 第1步：获取数据
并行调用：
1. `mcp__niuniuniu__stock_realtime` — 实时行情
2. `mcp__niuniuniu__stock_l2_dashboard` — L2数据（封单、大单、成交量）
3. `mcp__niuniuniu__stock_large_orders` — 大单统计
4. `mcp__niuniuniu__stock_timeshare` — 分时走势
5. `mcp__niuniuniu__limit_up_themes` — 涨停题材
6. WebSearch 搜索该股最新消息、连板情况、市场情绪

### 第2步：判断适用战法
根据股票当前状态和用户问题，判断适用哪个战法：
- 连板/大涨后首次收阴 → 龙头首阴低吸
- 放巨量封涨停 / 用户问"明天怎么操作/竞价怎么看/弱转强" → 爆量涨停&次日弱转强
- 当日首板涨停 / 用户问"哪些首板明天能二板/一进二" → 首板一进二

### 第3步：按战法框架分析

纯文本输出格式：

```
【{股票名}】{战法名}分析

一、战法条件检查
{逐项检查触发条件，打勾或打叉}

{支撑分析的核心数据}

三、情绪环境
{当前情绪周期 + 该战法在当前环境的胜率评估}

四、操作计划
买点: {具体价位或条件}
仓位: {X成}
止损: {具体价位和条件}
目标: {预期收益}

五、风险提示
{最大风险 + 应对}
```

## 注意事项
- 三大战法都有严格的情绪适配要求，退潮期原则上全部不做
- 不要同时满仓做多个战法，总仓位控制在6成以内
- 止损是铁律，没有"再等等看"的说法
